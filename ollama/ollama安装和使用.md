# Ollama下载和常见命令

## 下载

1. 官网下载安装包：ollama官网：https://ollama.com/

2. 配置环境变量。

注意：一般电脑显存是多大，差不多就能跑几B的模型。

**蒸馏模型**：是一种通过**知识蒸馏**技术对大模型进行轻量化压缩的产物，它像大模型的 “学生”，通过学习庞大教师模型的逻辑与能力，在保留核心功能的同时大幅缩减参数量，从而显著降低推理速度与显存占用，让大模型能力得以在普通电脑等低算力设备上部署运行。

## ollama常见命令

```sh
# 1. 运行一个模型（自动下载 + 启动）
ollama run qwen:7b

# 2. 查看本地已安装的模型
ollama list

# 3. 下载模型（不启动）
ollama pull qwen:7b

# 4. 删除不需要的模型
ollama rm qwen:7b

# 5. 查看正在运行的模型
ollama ps

# 6. 停止所有运行中的模型
ollama stop
```
